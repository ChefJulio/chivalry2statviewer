Chivalry 2 Stats Viewer

Load your FlavorStats save file to see every stat the game tracks.

How to find your stats file

  1. Copy this path: %LOCALAPPDATA%\Chivalry 2\Saved\Cloud
  2. Press Win+E to open File Explorer, paste the path into the address bar at the top, and press Enter.
  3. Find the files whose names start with FlavorStats_. If there is more than one, yours is the largest and most recently modified one that does not end in _local. (Smaller ones belong to other accounts that have played on your PC; _local files are the game's own working copies.)
  4. Drag that file onto the box above, or click the box and browse to it.

Your file is parsed locally in your browser and never uploaded.
